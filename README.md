# Health Risk Score Regressor Pipeline
This python script loads environmental features and trains Random Forest / Gradient Boosting Regressors.
These ML algorithms can predict the health risk score based on environmental features in the dataset. A separate script generates histograms and scatter plots of the data

Instructions:
- Run main.py from your IDE.
- (Optional) Run graphs.py from your IDE. Modify to change the data presented

Requirements:
- Python 3.14+
- At least 8GB RAM
  
Recommended:
- Python virtual environment
